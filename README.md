# Implementation of the Runge-Kutta method

## Summary

Numerical integration of Lotka-Volterra equations and the harmonic oscillator with the Runge-Kutta method and the Euler method.

## Usage

```
$ make
$ ./a.out
$ gnuplot plot.plt
```
